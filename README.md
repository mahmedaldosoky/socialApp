# Social App

#### Social media platform for Android and iOS built on Flutter and Firebase

### Features:
- Register and login through email authentication
- Real-Time Chat
- Making new friends
- Latest posts
- Adding new posts
- Update post
- Delete psot
- Likes
- Comments
- Personal Profile
- Updating your profile data
